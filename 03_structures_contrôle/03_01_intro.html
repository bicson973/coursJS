Structure de contrôle - intro

2 types de structures de contrôles de base

Les conditions et les boucles

2 types de structures de contrôles de base, que l'on retrouve dans tous les langages de programmation, et donc en JAVASCRIPT.

Les conditions

Les conditions vont nous permettre d'éxécuter un certain nombre d'instructions si seulement une certaine condition est vérifié.

Les boucles

Les boucles vont nous permttre d'éxécuter un bloc de condition en boucle tant qu'une condition donnée est vérifié.

Exemple : Dès que mes 12 élèves sont montés dans le bus je pars.

Opérateurs de comparaison

Comment vérifié les conditions ?

Dans le cas où la comparaison est vérifié, le JS renvoie le booléen true, dans le cas contraire, le booléen false est renvoyé.

Le JS va automatiquement comparer la valeur à gauche de l'opérateur à celle de droite selon l'opérateur de comparaison.
