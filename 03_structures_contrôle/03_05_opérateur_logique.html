Les opérateurs logiques

Ce type d'opérateurs va nous permettre d'éffectuer plusieurs comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test.

1/ Présentation et utilisation des opérateurs logiques.

Opérateurs logiques
Nom Symbol Description
AND (ET) && renvoie true si toutes les comparaisons sont vraies ou falsesinon.
OR (ou) || renvoie true si au moins l'une des comparaisons est vrai, ou falsesinon.
No (non) ! renvoie false si une comparaisons est évalué à true
