Structure de contrôle - if, else

IF (si)... ELSE (sinon)

1/ If...

La condition "if", "else" sera plus complête car on va exécuter un bloc de code si un test renvoie true ou un autre dans le cas contraire.

Attention !

Tout test d'une condition va être évalué dans contexte booléen ; donc à true à l'exception des valeurs suivantes, qui vont être évalué à false.

  • La valeur 0 (zéro)
  • le vooléen false
  • Une chaine de caractère vide ''
  • La valeur null
  • La valeur undefined
  • La valeur NaN(Not a Number)

Inverser la valeur logique d'un test

Le code placé dans le if n'est exécuté que si le résultat de la comparaison est false : on peut verifier que c'est faux en comparant le résultat de la comparaison à false.
