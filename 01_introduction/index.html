Le javascript est un langage dynamique

Ce langage va nous permettre de manipuler des contenus html ou des styles CSS et de les modifier en fonction de divers évènements ou variables. Un évènement peut être par exemple un clic d'un utilisateur à un certain endroit de la page tandis q'une variable peut être l'heure de la journée.

Le javascript est un langage principalement côté client

JS s'éxecute côté client, nous allons quand même pouvoir l'utiliser côté serveur à condition que l'on mette en place un environnement favorable, en utilisant, par ex. node.js.

Javascript est un langage interprété

Pour être interprété JS doit être éxécuté directement dans un logiciel inerpréteur : tous les naviguateurs connus possèdent leur interpréteur JS.

Javascript n'est pas JAVA !!!

Encore aujourd’hui, certaines personnes ont tendance à confondre les deux langages « Java » et « JavaScript ». Retenez ici que ces deux langages, bien que syntaxiquement assez proches à la base, reposent sur des concepts fondamentaux complètement différents et servent à effectuer des tâches totalement différentes.
